# LPC2129-UART-Motor-LED-Control

This project demonstrates UART interrupt-based control of an LED and a DC motor using LPC2129 with a 16x2 LCD.

## Features
- UART0 interrupt receive
- LED control (active low)
- DC Motor ON/OFF via L293D
- 16x2 LCD display
- Compatible with HC-05 Bluetooth module

## Commands
'LED ON'     → LED ON
'LED OFF'    → LED OFF
'MOTOR ON'   → Motor ON
'MOTOR OFF'  → Motor OFF

## Controller
LPC2129 (ARM7)
